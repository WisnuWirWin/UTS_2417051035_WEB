SelfForge

Kotlin Jetpack Compose Mobile App Productivity

Project Overview

SelfForge adalah aplikasi mobile produktivitas yang dibangun menggunakan Jetpack Compose, framework UI modern untuk pengembangan Android. Aplikasi ini dirancang untuk membantu pengguna membangun kebiasaan positif, mengelola tugas harian, dan meningkatkan produktivitas secara konsisten.

SelfForge memberikan pengalaman pengguna yang intuitif dan menyenangkan. Aplikasi ini memanfaatkan kemampuan declarative UI dari Jetpack Compose untuk menghadirkan antarmuka yang responsif dan smooth.

Technologies Used

  • Kotlin
  • Jetpack Compose
  • Android Studio
