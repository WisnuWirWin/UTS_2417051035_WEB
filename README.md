# Portofolio Wisnu

Website portofolio pribadi bertema **JoJo's Bizarre Adventure** yang menggabungkan estetika anime dengan desain modern.

##  Fitur Utama
- **JoJo Aesthetic**: Animasi "Menacing", Stand Stats, dan skema warna ikonik.
- **Projects**: Showcase proyek Minesweeper (C++), Sistem Rekomendasi (D&C), dan SelfForge (Jetpack Compose).
- **Interactive**: Validasi formulir kontak dan modal interaktif.

##  Stack
- HTML5, CSS3 (Animations), Vanilla JavaScript.
- C++, Kotlin, Jetpack Compose.
