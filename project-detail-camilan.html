Sistem Rekomendasi Paket Camilan

Algorithm Divide and Conquer Recommendation System

Project Overview

Sistem Rekomendasi Paket Camilan adalah sebuah program yang dirancang untuk membantu pelanggan menemukan paket camilan yang paling sesuai dengan preferensi mereka. Sistem ini menganalisis berbagai faktor seperti jenis rasa, kategori camilan, dan preferensi personal untuk menghasilkan rekomendasi yang akurat.

Proyek ini menerapkan pendekatan algoritma divide and conquer untuk memecah masalah rekomendasi menjadi sub-masalah yang lebih kecil, kemudian menggabungkan hasilnya (combine) untuk mendapatkan rekomendasi paket camilan yang optimal bagi setiap pelanggan.

Technologies Used

  • Divide and Conquer Algorithm
  • Data Processing
  • Sorting & Filtering
